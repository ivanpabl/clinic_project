# 🏥 Система онлайн-записи в поликлинику

![Django](https://img.shields.io/badge/Django-4.2-green)
![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.1-purple)

Веб-приложение для управления записями пациентов в поликлинике с разделением ролей (пациенты, врачи, администраторы).

## ✨ Основные функции

### 👥 Для пациентов:
- 📅 Онлайн-запись на прием к врачам
- 👨‍⚕️ Выбор специализации/услуги и врача
- 📱 Личный кабинет с историей записей
- ⭐ Оставление отзывов о врачах
- 🔔 Уведомления и напоминания

### 👨‍⚕️ Для врачей:
- 📊 Личный кабинет врача
- 📅 Управление расписанием и приемами
- 📈 Просмотр статистики
- 👥 Управление записями пациентов
- ✏️ Добавление медицинских заметок

### 🛠️ Административные функции:
- 👥 Управление пользователями и ролями
- 📝 Управление услугами и специализациями
- 📢 Публикация новостей
- 📊 Аналитика и отчеты
